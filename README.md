## English

### Overview
Project Asteria is a full-stack web application and dashboard designed to monitor Near-Earth Objects (NEOs). The platform not only tracks asteroid trajectories and calculates potential impact probabilities but also leverages Machine Learning to classify asteroid composition based on spectral data. By determining if an asteroid is Carbonaceous (C-type), Stony (S-type), or Metallic (M-type), the system estimates its potential commercial value in the space mining industry.

### Value Proposition
* **Scientific Outreach:** Provides accessible, real-time data on planetary defense.
* **Economic Forecasting:** Offers a unique perspective on space resources by estimating the raw material value of passing asteroids (e.g., Iron, Nickel, Platinum group metals).
* **Technical Showcase:** Demonstrates advanced capabilities in API integration, 3D data visualization, and applied Machine Learning.

### 🛠️ Tech Stack
* **Frontend:** Next.js (React), Tailwind CSS, Three.js (React Three Fiber), Recharts/Chart.js
* **Backend:** FastAPI (Python), PostgreSQL, Redis, Celery
* **Machine Learning:** Scikit-Learn

### 🙏 API Credits
* **[NASA JPL CNEOS API](https://cneos.jpl.nasa.gov/):** Orbital parameters, close-approach data, and impact risk assessment.
* **[NASA JPL Small-Body Database (SBDB) API](https://ssd.jpl.nasa.gov/tools/sbdb_lookup.html):** Physical parameters including albedo, diameter, and spectral classification.
* **Commodity Market APIs:** For fetching terrestrial commodity prices to calculate estimated market value.

## Español

### Descripción General
El Proyecto Asteria es un panel de control y aplicación web full-stack diseñado para monitorear Objetos Cercanos a la Tierra (NEOs). La plataforma rastrea las trayectorias de los asteroides y calcula sus probabilidades de impacto potencial, al mismo tiempo que aprovecha el Machine Learning para clasificar la composición de los asteroides usando datos espectrales (Carbonáceos, Rocosos o Metálicos) y estimar su valor comercial en la minería espacial.

### Propuesta de Valor
* **Divulgación Científica:** Datos accesibles en tiempo real sobre defensa planetaria.
* **Pronóstico Económico:** Perspectiva única sobre recursos espaciales mediante la estimación del valor en materias primas de los asteroides (Hierro, Níquel, metales del grupo del Platino).
* **Demostración Técnica:** Capacidades avanzadas en integración de interfaces (APIs), visualización de datos en 3D y Machine Learning.

### 🛠️ Tecnologías 
* **Frontend:** Next.js (React), Tailwind CSS, Three.js (React Three Fiber), Recharts/Chart.js
* **Backend:** FastAPI (Python), PostgreSQL, Redis, Celery
* **Machine Learning:** Scikit-Learn

### 🙏 Créditos de APIs
* **[NASA JPL CNEOS API](https://cneos.jpl.nasa.gov/):** Parámetros orbitales, datos de aproximación y riesgo de impacto.
* **[NASA JPL Small-Body Database (SBDB) API](https://ssd.jpl.nasa.gov/tools/sbdb_lookup.html):** Parámetros físicos (albedo, diámetro y clasificación espectral).
* **Commodity Market APIs:** Para obtener los precios terrestres de las materias primas y calcular el valor del asteroide.
